# 정산 용어 안내

## 📌 이 문서는?

정산 페이지에서 사용되는 용어들의 의미를 쉽게 설명한 문서입니다.

## ⏰ 정산 시점 (중요!)

**정산은 항상 익월에 이루어집니다.**

- 1월의 계약 건과 출고 건 → **2월 정산**에서 확인 가능
- 2월의 계약 건과 출고 건 → **3월 정산**에서 확인 가능
- 3월의 계약 건과 출고 건 → **4월 정산**에서 확인 가능

**예시**: 1월에 계약 5건, 출고 3건을 달성했다면, 이 실적은 2월 정산 페이지에 나타나며 2월에 지급됩니다.

---

## 🚗 계약 관련 용어

계약은 **고객 상태가 '계약'으로 잡힌 건들**을 말합니다.

| 용어              | 의미                       | 설명                                            |
| ----------------- | -------------------------- | ----------------------------------------------- |
| **계약 건수**     | 계약을 체결한 건수         | 해당 월에 몇 건의 계약을 했는지                 |
| **계약 매출**     | 계약으로 발생한 총 매출액  | 계약 건들의 매출을 모두 합친 금액               |
| **계약 지출**     | 계약을 위해 사용한 총 비용 | 계약 건들의 지출을 모두 합친 금액               |
| **계약 순익**     | 계약으로 남은 순수 이익    | 계약 매출에서 계약 지출을 뺀 금액 (매출 - 지출) |
| **계약 수당** | 계약으로 받는 수당         | (계약 매출 - 계약 지출) × 직책별 수당율로 계산<br/>**※ 실제 지급에는 영향이 없으며, 참고용으로만 표시됩니다** |

### 예시

- 계약 건수: 5건
- 계약 매출: 5,000만원
- 계약 지출: 4,500만원
- **계약 순익: 500만원** (5,000만원 - 4,500만원)
- **계약 수당: 25만원** (500만원 × 5%)
  - ※ 참고용으로만 표시되며, 실제 급여에는 반영되지 않습니다

## 🚚 출고 관련 용어

출고는 **고객과 실제 계약을 하고 차량까지 출고가 된 건들**을 말합니다.

| 용어              | 의미                       | 설명                                            |
| ----------------- | -------------------------- | ----------------------------------------------- |
| **출고 건수**     | 차량을 출고한 건수         | 해당 월에 몇 대의 차량을 출고했는지             |
| **출고 매출**     | 출고로 발생한 총 매출액    | 출고 건들의 매출을 모두 합친 금액               |
| **출고 지출**     | 출고를 위해 사용한 총 비용 | 출고 건들의 지출을 모두 합친 금액               |
| **출고 순익**     | 출고로 남은 순수 이익      | 출고 매출에서 출고 지출을 뺀 금액 (매출 - 지출) |
| **출고 수당** | 출고로 받는 수당           | (출고 매출 - 출고 지출) × 직책별 수당율로 계산<br/>**※ 실제 지급되는 수당 금액입니다** |

### 예시

- 출고 건수: 3건
- 출고 매출: 3,000만원
- 출고 지출: 2,700만원
- **출고 순익: 300만원** (3,000만원 - 2,700만원)
- **출고 수당: 15만원** (300만원 × 5%)
  - ※ 실제 급여에 반영되는 수당 금액입니다

## 💰 계약과 출고의 차이

| 구분          | 계약                             | 출고                                   |
| ------------- | -------------------------------- | -------------------------------------- |
| **상태**      | 고객 상태가 '계약'으로 설정된 건 | 실제 계약을 하고 차량까지 출고된 건    |
| **의미**      | 계약 단계                        | 계약 + 차량 출고 완료                  |
| **정산 기준** | 고객 상태가 계약으로 변경된 월의 **익월** 정산 | 차량이 실제로 출고된 월의 **익월** 정산 |
| **수당 지급** | 참고용으로만 표시 (실제 지급 없음) | 실제 급여에 반영되어 지급됨 |

### 예시로 이해하기

```
1월: 홍길동 고객의 상태를 '계약'으로 변경 → 2월 정산에 "계약" 항목으로 반영
3월: 홍길동 고객과 실제 계약 체결 및 차량 출고 → 4월 정산에 "출고" 항목으로 반영
```

**중요**: 정산은 익월에 이루어지므로, 1월 실적은 2월 정산에서 확인할 수 있습니다.

## 💵 기타 수당

### 1. 추가 수당

- **의미**: 특별히 지급되는 추가 수당
- **입력 방법**: 정산 페이지에서 관리자가 직접 입력
- **활용 용도**: 관리자가 판단 (성과급, 특별 수당 등 다양한 목적으로 사용 가능)

### 2. 기타 수당

- **의미**: 팀장, 본부장 등이 받는 팀 전체 성과 수당
- **계산 방법**:
  - **팀장**: (본인의 출고 순익 × 관리자가 설정하는 직책별 수당율) + (팀매출 × 팀매출 수당율)
  - **본부장**: 본부매출 × 본부매출 수당율
  - 비율은 관리자 페이지의 "본부별 수당" 메뉴에서 설정 가능
- **자동 계산**: 시스템이 자동으로 계산

**계산 예시**
- **팀장**: 본인의 출고 순익이 500만원이고, 직책별 수당율이 20%, 팀매출이 1억원, 팀매출 수당 비율이 10%로 설정된 경우
  - 기타 수당 = (500만원 × 20%) + (1억원 × 10%) = 100만원 + 1,000만원 = 1,100만원
- **본부장**: 본부매출이 5억원이고, 본부매출 수당 비율이 5%로 설정된 경우
  - 기타 수당 = 5억원 × 5% = 2,500만원

### 3. 상여금

- **의미**: 명절이나 특별한 경우에 지급되는 보너스
- **입력 방법**: 정산 페이지에서 관리자가 직접 입력
- **활용 용도**: 관리자가 판단 (명절 상여금, 회사 경영 성과에 따른 보너스 등)

## 📊 총 수당 계산 방법

### 일반 영업사원

```
총 수당 = 출고 수당 + 추가 수당 + 상여금
```

**예시**

- 출고 수당: 15만원
- 추가 수당: 10만원
- 상여금: 50만원
- **총 수당: 75만원**

### 팀장

```
총 수당 = 출고 수당 + 기타 수당 + 추가 수당 + 상여금
```

**예시**

- 출고 수당: 15만원
- 기타 수당: 1,100만원 ((본인 출고 순익 × 직책별 수당율) + (팀매출 × 팀매출 수당율))
- 추가 수당: 10만원
- 상여금: 50만원
- **총 수당: 1,175만원**

### 본부장

```
총 수당 = 출고 수당 + 기타 수당 + 추가 수당 + 상여금
```

**예시**

- 출고 수당: 15만원
- 기타 수당: 2,500만원 (본부매출의 5%)
- 추가 수당: 10만원
- 상여금: 50만원
- **총 수당: 2,575만원**

## 🔍 용어 빠른 찾기

### ㄱ

- **계약**: 고객 상태가 '계약'으로 잡힌 건들
- **계약 건수**: 체결한 계약의 개수
- **계약 매출**: 계약으로 발생한 매출 금액
- **계약 지출**: 계약을 위해 사용한 비용
- **계약 순익**: 계약 매출에서 지출을 뺀 순수 이익
- **계약 수당**: (계약 매출 - 계약 지출) × 직책별 수당율 (실제 지급에는 영향이 없으며, 참고용으로만 표시)
- **기타 수당**: 팀장은 (본인 출고 순익 × 직책별 수당율) + (팀매출 × 팀매출 수당율), 본부장은 본부매출 × 본부매출 수당율 (비율은 관리자 페이지의 "본부별 수당"에서 설정, 자동 계산)

### ㅅ

- **상여금**: 관리자가 직접 입력하는 보너스 (활용 용도는 관리자 판단)
- **순익**: 매출에서 지출을 뺀 순수 이익

### ㅊ

- **출고**: 고객과 실제 계약을 하고 차량까지 출고가 된 건들
- **출고 건수**: 인도한 차량의 개수
- **출고 매출**: 출고로 발생한 매출 금액
- **출고 지출**: 출고를 위해 사용한 비용
- **출고 순익**: 출고 매출에서 지출을 뺀 순수 이익
- **출고 수당**: (출고 매출 - 출고 지출) × 직책별 수당율 (실제 지급되는 수당 금액)
- **추가 수당**: 관리자가 직접 입력하는 추가 수당 (활용 용도는 관리자 판단)

### ㅇ

- **수당**: 성과에 따라 받는 금액

## 💡 자주 묻는 질문

### Q1. 기타 수당은 누가 받나요?

**A.** 팀장과 본부장 등 관리직이 받습니다. 팀원들의 성과를 관리한 것에 대한 수당이며, 수당 비율은 관리자 페이지의 "본부별 수당" 메뉴에서 설정할 수 있습니다.

### Q2. 추가 수당과 상여금의 차이는?

**A.** 둘 다 관리자가 직접 입력하는 항목이며, 활용 용도는 관리자가 판단합니다.

- **추가 수당**: 특별한 성과나 업무에 대한 수당 등
- **상여금**: 명절 상여금, 회사 경영 성과에 따른 보너스 등

실제 사용 목적은 관리자가 필요에 따라 유연하게 결정할 수 있습니다.

### Q3. 정산은 언제 이루어지나요?

**A.** 정산은 항상 익월에 이루어집니다.

- 1월의 계약과 출고 실적 → 2월 정산
- 2월의 계약과 출고 실적 → 3월 정산
- 3월의 계약과 출고 실적 → 4월 정산

**예시**: 1월 한 달 동안의 업무 성과는 2월에 정산되어 확인하고 지급받을 수 있습니다.

### Q4. 왜 익월에 정산하나요?

**A.** 해당 월의 모든 업무가 완료되고 데이터가 정리된 후 정확한 정산을 하기 위해 익월에 정산합니다. 이를 통해:
- 해당 월의 모든 계약과 출고 건이 누락 없이 반영됩니다
- 정확한 매출과 지출 집계가 가능합니다
- 오류 없는 정산을 보장할 수 있습니다
